# 眠れぬ夜に for Android
安らかな眠りを支援するスマホアプリ

もともとは1991年に作ったMS-DOSの常駐ソフトで、それを1995年にWindows3.1用のソフトウェアとして作り直し、さらに2021年の今、スマホアプリとして作り直したのでした。
